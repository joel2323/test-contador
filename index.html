¿Qué es "Hola Mundo"?

"Hola Mundo" es el programa más básico que se suele crear cuando se aprende un nuevo lenguaje de programación. Es una tradición entre desarrolladores que se remonta a los años 70. En HTML, crear un "Hola Mundo" significa construir tu primera página web que muestre este saludo.
